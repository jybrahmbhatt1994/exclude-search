A plugin to exclude specific posts/pages/product/custom post from WordPress search results.

== Description ==

The Exclude Search plugin allows you to exclude specific posts, pages, and custom post types (including WooCommerce products) from WordPress search results. 
This plugin provides an intuitive admin interface where you can manage the exclusion of posts from appearing in search results, making it easier to fine-tune your site's search functionality.

== Features ==

- Exclude specific posts/pages/custom post types from WordPress search results.
- Exclude WooCommerce products from search results (if WooCommerce is installed).
- Easy-to-use admin interface for managing excluded items.
- Option to search through posts, pages, and custom post types in the admin panel.
- Pagination support for large datasets.
  
== Installation ==

1. Upload the `exclude-search` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Once activated, a new menu item named "Exclude Search" will appear in the WordPress admin menu.
4. Use the admin page to search posts, pages, and custom post types. You can select specific items to exclude from search results.

== How to Use ==

1. Go to the "Exclude Search" menu in the admin dashboard.
2. Select a post type (Posts, Pages, Products, or Custom Post Types).
3. Use the search box to find the posts you want to exclude.
4. Select the posts you wish to exclude from search results.
5. Click "Save Changes" to update your exclusions.

The excluded posts will no longer appear in the WordPress search results for your site visitors.

== Frequently Asked Questions ==

= How do I exclude posts from search results? =
1. Go to the "Exclude Search" menu in your WordPress admin.
2. Select the post type (Post, Page, Product, etc.).
3. Search for the posts you want to exclude.
4. Check the boxes next to the posts and click "Save Changes."

= Can I exclude WooCommerce products from search? =
Yes, if you have WooCommerce installed, you can exclude products from search results by selecting "Products" in the post type tab.

= What happens if I remove posts from the excluded list? =
If you remove a post from the excluded list, it will appear in the search results again.
